Simulation of the given IPL Match.
MapReduce on Hadoop was used to carry out the simulation along with Hive.

File Details -
	IPLSimulation.java - Contains the code for the IPL match simulator
	SampleInput.txt - Contains the batting and bowling order for each team for input to Hive
	QueriesForClusterNums.txt - Contains the queries required to obtain the cluster numbers for each player in the match
	clusterno.txt - Contains the cluster numbers for every player in their correct playing order for input to MapReduce
